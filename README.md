# 🌐 Word Explorer App
Word Explorer App is a simple web application that lets you explore any word and see related images. Enter a word, and the app will:  
Find similar words  
 Show related images  

This project is perfect for learning, brainstorming, or just having fun discovering words and visuals.  


## 📝 Features
🔍 Search for any word  
✏️ Find similar words using the **Datamuse API**  
🖼️ See related images using the **Pexels API**  
⚡ Fast, interactive, and easy to use  

## 🛠️ Technologies & APIs Used
Python – programming language  
Streamlit– for creating the web app interface  
Requests library – to fetch data from APIs  
Datamuse API – provides similar words and synonyms  
Pexels API – provides free stock images related to the word
